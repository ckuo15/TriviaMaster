# Trivia Master

#### To run on local machine, please run rails s, npm run webpack, and navigate to http://localhost:3000/#/

## [Live Link](http://trivia-masterr.herokuapp.com/#/)

### Technologies used: 
  - Ruby on Rails
  - PostgreSQL
  - JavaScrpt
  - React
  - Redux
  - BCrypt
  - HTML5
  - SASS
  - jQuery
  
 ## Features:
  * User authentication
  * Question and answer creation
  * Categories selection
